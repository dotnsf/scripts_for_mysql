# scripts_for_mysql

Shell scripts to handle MySQL by cli.


## How to use

### create_db.sh

- `$ MYSQL_ROOT_PASSWORD=xxxxx ./create_db.sh`


### drop.sh

- `$ MYSQL_ROOT_PASSWORD=xxxxx ./drop_db.sh [dbname]`
